# Large Software System Design

A collection of software design artifacts produced as part of a large-scale systems design course at McMaster University.

## Contents

- **System Architecture** — High-level system architecture diagram
- **Analysis Class Diagram** — UML analysis class diagram (Deliverable 2)
- **Model-View-Controller** — MVC design pattern description
- **Observer Pattern** — UML class diagram for the Observer design pattern
- **Sorting Algorithm Time Complexity** — Best, worst, and average case complexity comparison
- **Verification and Validation Model** — V-model for software verification and validation
